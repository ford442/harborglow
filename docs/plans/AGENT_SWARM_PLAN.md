# HarborGlow: Agent Research Swarm Plan

## Mission
Deep-dive research into real-world science behind ships, weather, wildlife, and their interactions. Generate authentic, fascinating details that elevate the simulation beyond arcade into "satisfying realism."

## Research Domains

### Domain 1: Naval Architecture & Ship Physics
**Lead:** Agent Shipwright
**Scope:** Real ship specifications, buoyancy calculations, stability curves, propulsion systems
**Deliverables:**
- Actual dimensions/tonnages for each ship class
- Metacentric height (GM) ranges by vessel type
- Propeller configurations and Kort nozzles
- Anchor types (stockless, Bruce, Danforth) and holding power
- Mooring line materials (nylon vs. chain vs. Dyneema)
- Ballast water systems and stability
- Cranes: spreader mechanics, twistlocks, Safe Working Load (SWL)

### Domain 2: Marine Meteorology
**Lead:** Agent Stormchaser
**Scope:** Real weather phenomena, forecasting, maritime impacts
**Deliverables:**
- Beaufort scale correlation with wave heights
- Fog formation mechanisms (radiation vs. advection vs. sea smoke)
- Lightning detection systems on ships (loop antennas)
- Waterspout formation (fair weather vs. tornadic)
- Tropical cyclone categories and harbor protocols
- Rogue wave science (Draupner wave, nonlinear Schrödinger)
- Ice accretion rates on vessels (Icing Intensity Scale)

### Domain 3: Marine Biology & Ecology
**Lead:** Agent Biologist
**Scope:** Real animal behaviors, ecosystems, human interactions
**Deliverables:**
- Whale diving profiles (sperm whales: 90min @ 2000m)
- Bioluminescence mechanisms (dinoflagellates, bacteria)
- Polar bear hunting ranges and sea ice dependency
- Dolphin echolocation frequencies and ranges
- Deep sea pressure adaptations (1200 atm at Hadal zone)
- Migration corridors and shipping lane conflicts
- Biofouling organisms (barnacles, tubeworms) and ship economics

### Domain 4: Maritime Operations & Safety
**Lead:** Agent Mariner
**Scope:** Real procedures, equipment, emergency protocols
**Deliverables:**
- COLREGs (Collision Regulations) light configurations
- AIS (Automatic Identification System) transponder classes
- GMDSS (Global Maritime Distress) communication hierarchy
- Pilot ladder arrangements and boarding procedures
- Dynamic Positioning (DP) system classes (1/2/3)
- Firefighting on ships (CO2 flooding, water mist, foam)
- Search and rescue patterns (expanding square, sector search)

### Domain 5: Specialized Vessel Deep-Dives
**Lead:** Agent Specialist
**Scope:** Unique ship types and their fascinating specifics
**Deliverables:**
- Fireboat pump capacities (20,000+ GPM monsters)
- Icebreaker hull designs (round vs. spoon bow)
- Submarine rescue chambers and McCann bells
- Cable layer tensioning (2-10 ton range, delicate!)
- Hospital ship Geneva Convention protections
- Aircraft carrier arresting gear (Mk7 hydraulic)
- Research vessel A-frames and winch systems

### Domain 6: Port Infrastructure & Geography
**Lead:** Agent Geographer
**Scope:** Real harbors, geography, unique features
**Deliverables:**
- Fjord geology and navigation challenges
- Atoll lagoon dynamics and entry passages
- Iceberg classification (growler, bergy bit, small/medium/large)
- Tidal ranges by location (Bay of Fundy: 16m!)
- Breakwater designs (rubble mound vs. caisson)
- Container terminal crane types (ship-to-shore, RTG, straddle)
- Panama Canal lock filling (3 pools, gravity-fed)

## Output Format
Each agent produces a markdown file with:
1. **Facts** — Bullet-point verified information with sources
2. **Fascinating Details** — The "wait, really?" moments
3. **Simulation Applications** — How this becomes gameplay
4. **Source Links** — For verification and deeper reading

## File Structure
```
research/
├── naval_architecture.md
├── meteorology.md
├── marine_biology.md
├── maritime_operations.md
├── specialized_vessels.md
├── port_geography.md
└── synthesis.md (combined insights)
```

## Success Criteria
- Every ship stat grounded in real vessels
- Weather effects based on actual physics
- Wildlife behaviors match scientific literature
- Operations reflect real maritime procedures
- At least 5 "I didn't know that" moments per domain
