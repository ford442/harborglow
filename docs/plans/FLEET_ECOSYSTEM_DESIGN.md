## 🗺️ Destination Themes

### Tier 1: Paradise / Commercial
**Vibe:** Vacation brochures, retirement dreams, Instagram filters

**Primary Ships:**
- Mega Cruise Liner
- Superyacht
- Ferry (tourist routes)
- Fishing Trawler (recreational)

**Wildlife:**
- Dolphins (bow-riding)
- Sea Turtles
- Manta Rays
- Tropical fish schools
- Frigatebirds

**Weather:**
- Clear Starfield
- Golden Hour
- Bioluminescent Bloom
- Light Fog (atmospheric)

**Signature Event:** *Sunset Sail* — cruise ships depart at golden hour with full light show

**Music Palette:** Upbeat, vacation-core, steel drums, synth-pop

## 🎲 Dynamic Event System

### Random Encounters
| Event | Frequency | Effect |
|-------|-----------|--------|
| **Whale Breach** | Uncommon | Spectacle, pauses operations briefly |
| **School of Dolphins** | Common | Bow-riding show, morale boost |
| **Oil Spill Detected** | Rare | Pollution control vessel deploys |
| **Distress Call** | Uncommon | Coast Guard response, rescue operation |
| **VIP Arrival** | Rare | Superyacht with security, restricted zone |
| **Military Exercise** | Rare | Navy ships, temporary no-go areas |
| **Iceberg Drift** | Arctic only | Icebreaker dispatch, route changes |
| **Bioluminescent Event** | Seasonal | Tourism boom, photo opportunities |

### Chain Reactions
1. **Storm Warning** → Ships secure operations → Emergency vessels standby → Wildlife flees
2. **Ship Fire** → Fireboat deploys → Coast Guard cordon → Hospital ship alert → Media helicopters
3. **Iceberg Calving** → Tsunami warning → Ships evacuate → Research vessels document → Icebreaker crushes
4. **Whale Stranding** → Rescue vessels → Scientists → Crowds → Tragic/failure outcomes possible
